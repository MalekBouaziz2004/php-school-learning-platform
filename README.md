PHP School Learning Platform
📚 Project Overview

This project is a web-based School Learning Platform developed using PHP and MySQL.
It allows students and teachers to manage learning materials, exercises, and subject-related content in a structured way.

👥 Team Project

This application was developed as part of a team assignment (Teamarbeit) during the Software Engineering module.
The goal was to design and implement a functional web application following software development best practices.

🚀 Features
🔐 User authentication (login/logout system)
📂 Upload and manage learning materials
📖 Subject and category management (admin)
📝 Exercises and explanations system
🔍 Keyword search functionality
📩 Contact and messaging system
🏗️ Project Structure
assets/        # CSS, JS, images
auth/          # Authentication system
config/        # Database connection
includes/      # Shared layout components
modules/       # Core features (subjects, exercises, explanations)
admin/         # Admin functionalities
public/        # Entry point (index.php)
database/      # SQL setup
uploads/       # User uploaded files
⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/php-school-learning-platform.git
Move the project to your local server (e.g., XAMPP htdocs)
Import the database:
Open phpMyAdmin
Import database/setup.sql
Configure database connection:
Edit config/db_connect.php

Run the project:

http://localhost/php-school-learning-platform/public/
